# Product Requirements Document

## Goal

Build an AI-powered internship interview platform for agriculture domain students.

## Features

1. User authentication
2. Resume upload
3. Candidate profile form
4. AI interview chatbot
5. Evaluation and scoring
6. Auto-generated summary
7. FAQ support bot
8. Human escalation
9. Recruiter dashboard
10. Anti-cheating measures

## User Flow

Signup/Login
→ Upload Resume
→ Fill Candidate Form
→ AI Interview
→ Evaluation
→ Result
→ Offer/Reject Email
